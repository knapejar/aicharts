aicharts

Professional charts for AI agents

One tool, any chart, beautiful defaults. Rendered as PNG, ready for chat, reports, slides.

Try it Install GitHub

Install

Claude Desktop

Add to claude_desktop_config.json:

{
  "mcpServers": {
    "aicharts": {
      "command": "npx",
      "args": ["-y", "aicharts"]
    }
  }
}

Claude Code / Cursor

claude mcp add aicharts -- npx -y aicharts

ChatGPT

Settings → Connectors → Developer Mode → Add HTTP URL:

https://mcp-charts.vercel.app/mcp

Or just call the HTTP API: POST https://mcp-charts.vercel.app/chart with a JSON body.

For AI agents

        

          

            
Ask Claude or ChatGPT

            
“Using render_chart, make a bar chart of my last 4 quarter revenue.”

            
The model picks a chart type, formats the data, and calls the tool. You get a PNG inline.

          

          

            
Or call the HTTP API yourself

            
curl -X POST https://mcp-charts.vercel.app/chart \
  -H "Content-Type: application/json" \
  -d '{"chart":"bar","data":[{"label":"Q1","value":120},{"label":"Q2","value":180}],"title":"Revenue"}' \
  --output chart.png
